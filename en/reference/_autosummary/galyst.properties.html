galyst.properties

galyst.properties#

This package provides a collection of property calculators for use with the tangos framework. These classes define how to calculate specific physical properties for simulation objects (e.g., halos, groups).

When imported, these calculators will be identified automatically, and their results can be written to the database, making them available for later analysis.

Each class inherits from tangos.properties.pynbody.PynbodyPropertyCalculation and typically defines:

names

String or a tuple of property names that will be created in the database.

calculate()

The core method that performs the calculation.

region_specification()

A method that defines the region of the simulation to load for the calculation.

requires_property()

A method that lists any prerequisite properties.

preloop():

Perform one-per-snapshot calculations, given the loaded simulation data and TimeStep object.
